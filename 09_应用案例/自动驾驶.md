## 1. 自动驾驶概述

### 1.1 分级

| 级别 | 名称 | 职责 |
|------|------|------|
| L0 | 无自动化 | 人类驾驶 |
| L1 | 辅助驾驶 | 方向或速度控制 |
| L2 | 部分自动化 | 方向+速度 |
| L3 | 条件自动化 | 特定场景自动驾驶 |
| L4 | 高度自动化 | 大部分场景无需人类 |
| L5 | 完全自动化 | 全场景自动驾驶 |

## 4. 规划与控制

### 4.1 运动规划

```python
# 行为规划
class BehaviorPlanner:
    def plan(self, env_state):
        # 状态机选择
        if env_state.crossing_pedestrian:
            return "STOP"
        elif env_state.green_light:
            return "GO"
        else:
            return "WAIT"
```
