## 📚 关于本仓库

本仓库系统记录物理AI（具身智能）领域的学习内容，由浅入深，适合初学者到进阶。

### 什么是物理AI / 具身智能？

**物理AI（Physical AI）** 或 **具身智能（Embodied AI）** 是具有物理实体、能够感知理解并与世界交互的智能系统。

```

├── 03_感知技术/      # 计算机视觉、深度感知、传感器融合
├── 04_运动控制/      # 运动规划、轨迹跟踪、控制算法
```
