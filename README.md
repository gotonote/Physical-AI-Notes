## 📚 关于本仓库

本仓库系统地记录物理AI（具身智能）领域的学习笔记，从基础概念到前沿技术，旨在为初学者提供清晰的学习路径，为从业者提供技术参考。

### 什么是物理AI / 具身智能？

**物理AI（Physical AI）** 或 **具身智能（Embodied AI）** 是指具有物理实体、能够在真实或模拟环境中感知、理解并与世界进行交互的智能系统。

核心思想：**智能源于身体与环境的持续交互**。

## 🚀 快速开始

如果你是初学者，建议按以下顺序阅读：

1. **[01_基础概念](./01_基础概念/README.md)** - 建立整体认知
2. 数学与编程基础（线性代数、Python、PyTorch）
3. ROS基础与仿真环境搭建
4. 选择一个方向深入（感知/规划/学习）
