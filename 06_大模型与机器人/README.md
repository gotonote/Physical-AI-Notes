## 核心概念

### 大模型+机器人

| 技术 | 应用 |
|------|------|
| LLM | 任务规划、代码生成 |
| VLM | 视觉理解、指令解析 |
| VLA | 端到端控制 |

1. **RT-2**: Google 视觉-语言-动作模型
2. **PaLM-E**: Google 多模态大模型
3. **ALOHA**: 低成本开源机器人平台
