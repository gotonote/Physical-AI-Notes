## 1. 夹爪

### 1.1 常见类型

| 类型 | 特点 | 适用场景 |
|------|------|----------|
| 平行夹爪 | 简单可靠 | 规则物体 |
| 三指夹爪 | 多角度抓取 | 不规则物体 |
| 真空吸盘 | 平面吸附 | 表面平整物体 |
| 软体夹爪 | 适应性 | 软物体 |

### 1.2 控制示例

```python
# Robotiq 2F夹爪
from robotiq_2f_gripper import Robotiq2FGripper

gripper = Robotiq2FGripper()
gripper.activate()
gripper.set_position(100)  # 0-255
gripper.set_force(50)     # 0-255
```

## 2. 传感器

### 2.1 视觉传感器

```python
# RealSense D435
import pyrealsense2 as rs

pipeline = rs.pipeline()
config = rs.config()
config.enable_stream(rs.stream.depth, 640, 480, rs.format.z16, 30)
config.enable_stream(rs.stream.color, 640, 480, rs.format.bgr8, 30)

pipeline.start(config)

# 获取帧
frames = pipeline.wait_for_frames()
depth = frames.get_depth_frame()
color = frames.get_color_frame()
```

### 2.2 力传感器

```python
# ATI Force/Torque Sensor
from ati_ft import FTSensor

sensor = FTSensor(device='/dev/ttyUSB0')
ft_data = sensor.read()  # [Fx, Fy, Fz, Tx, Ty, Tz]
```

## 3. 选型指南

### 3.1 选择原则

1. **任务需求**: 抓取/操作 → 选择对应夹爪
2. **精度要求**: 高精度 → 力传感器+视觉
3. **环境**: 黑暗 → 深度相机+红外
4. **成本**: 预算限制 → 平衡功能与成本
