### 1.2 控制接口

```python
# 手术机器人控制
class SurgicalRobot:
    def teleoperate(self, master_pos, master_orien):
        # 映射到从端
        slave_pos = self.transform(master_pos)
        slave_orien = self.transform(master_orien)
        
        # 逆运动学求解
        q = self.ik(slave_pos, slave_orien)
        
        # 执行
        self.joint_control(q)
```
