## 📦 常用工具

### 框架库
- [LeRobot](https://github.com/huggingface/lerobot) - HuggingFace机器人学习框架
- [PyTorch](https://pytorch.org/) - 深度学习框架
- [Isaac Sim](https://developer.nvidia.com/isaac-sim) - NVIDIA仿真平台

### 仿真环境
- [MuJoCo](https://mujoco.org/) - 物理仿真引擎
- [PyBullet](https://pybullet.org/) - Python机器人仿真
- [Gymnasium](https://gymnasium.farama.org/) - 强化学习环境

### 机器人中间件
- [ROS 2](https://www.ros.org/) - 机器人操作系统
- [MoveIt](https://moveit.ros.org/) - 运动规划框架
